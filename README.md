# TomTomThis is a repo for managing machine readable artifacts for the [TomTom APIs](http://business.tomtom.com/en_us/solutions/integration/resources/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [TomTom APIs](http://business.tomtom.com/en_us/solutions/integration/resources/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://business.tomtom.com/en_us/solutions/integration/resources/)- [Developer](http://business.tomtom.com/en_us/solutions/integration/resources/)- [Twitter](https://twitter.com/TomTom)- [GitHub](https://github.com/tomtom)The [APIs.json](https://github.com/api-evangelist/tomtom/blob/master/apis.json) index provides links to all the relevant links for the TomTom APIs, including the OpenAPI and Postman if they are present.
